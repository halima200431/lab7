# ⭐ StarsGallery

Application Android affichant une galerie de stars avec notes et recherche.

## Stack
- Java · Android SDK 21+
- Glide · RecyclerView · CircleImageView

## Fonctionnalités
- Splash screen animé
- Liste avec photos circulaires et notes
- Recherche par nom en temps réel
- Modification de note via popup
- Partage de l'application
